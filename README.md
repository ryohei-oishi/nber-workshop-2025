# Spring 2025 NBER workshop
This repo will provide materials and code for the Spring 2025 heterogeneous-agent macro workshop at the NBER. See the [workshop NBER page for an agenda](https://www.nber.org/conferences/heterogeneous-agent-macroeconomics-workshop-spring-2025).

While this repo is under construction, feel free to check out the [repo for the 2023 workshop](https://github.com/shade-econ/nber-workshop-2023).
